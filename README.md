# WA TXT Parser Web

Sistem web untuk mengolah data mentah hasil ekspor chat WhatsApp (.txt) menjadi file Excel terstruktur secara otomatis.

Aplikasi ini dibangun untuk membantu admin memproses data dalam jumlah besar dengan format yang konsisten, cepat, dan akurat.

## Tech Stack
- Laravel (Web Interface)
- FastAPI (Parsing Engine)
- MySQL (Database)
- Python (Data Processing)

## Features
- Upload file chat WhatsApp (.txt)
- Parsing otomatis menjadi data terstruktur
- Export hasil ke Excel (.xlsx)
- Multi-admin support
- REST API (FastAPI)
- Processing history (planned)
